| Boost | ✅ Yes |

## How to Check

1. Look for LICENSE or LICENSE.md file in repo root
2. Check GitHub's license detection (shown on repo page)
3. Look for SPDX identifier in package.json, pyproject.toml, etc.

**If unclear:** Contact Shipd team before accepting.
